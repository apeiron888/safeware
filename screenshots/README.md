This directory contains screenshots of the SafeWare application.

2. **manager-dashboard.png** - Manager dashboard
3. **manager-warehouses.png** - Manager warehouses page
4. **manager-items.png** - Manager items page

## Screenshot Guidelines

- Format: PNG or JPG
- Resolution: At least 1920x1080 for desktop views
- Include: Full page view with sidebar visible
- Ensure: No sensitive data is visible in screenshots
